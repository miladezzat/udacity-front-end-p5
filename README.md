# project 5 travel app

1. Starts a dev server for the webpage
```javascript
    npm run build-dev 
```
2. creates a production build in the dist folder 
```javascript    
    npm run build-prod 
```
3. runs jest tests
```javascript
    npm run test     
```
4. starts the node server
```javascript
    npm run start 
```

4. start app
```javascript
    npm run app
```
